Introduction

This is a WeiDU mini-mod that restores the classic movies from the original Baldur’s Gate and select movies from Baldur’s Gate II to Baldur’s Gate: Enhanced Edition. It now also re-enables the Beregost movie when you first travel to the area. This mod was developed for Baldur's Gate Enhanced Edition v1.0.2007, and will be considered a beta version as long as BG:EE is being updated. Use at your own risk! Questions, comments, suggestions, and bug reports are of course welcome.

Installation

First of all, please read Player how-to: Getting mods to work on BGEE.

This is a WeiDU mod. To install, unzip the RAR archive into your game folder that has the chitin.key file (usually "Baldur's Gate Enhanced Edition\Data\00766") and double-click on Setup-BGEEClassicMovies.exe. To uninstall, double-click on BGEEClassicMovies.exe again and select the Uninstall option.

Additionally, you will need to edit your Baldur.ini file by hand. This is necessary in order to play the movies not used by BG:EE in the Movies GUI dialog. It is usually located at "Documents\Baldur's Gate - Enhanced Edition\Baldur.ini" in Win 7 and at "My Documents\Baldur's Gate - Enhanced Edition\Baldur.ini" in Win XP. Frequently Asked Questions

Q: Why does MOVIDESC.2DA and Baldur.ini need to be modified?
A: The Baldur.ini and MOVIDESC.2DA are modified so that all of the movies can be viewed in the "Movies" dialog of the GUI. Since not all of the movies from BG1 are actually used in BGEE, this is the only way to watch them.
